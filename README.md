# Detectior

Программное обеспечение для распознавания людей в кадре и отправки дальнейших событий.


## Dependencies

* [OpenCV](https://opencv.org)


## Quick start

```console
$ cc -o nob nob.c
$ nob
$ main
```


## Neural model

Нейронная модель была взята отсюда — https://github.com/J3lly-Been/YOLOv8-HumanDetection/
